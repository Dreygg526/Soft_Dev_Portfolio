📸  PHOTOS GO IN THIS FOLDER
============================================================

Drop your image files here (or in the /projects subfolder) and the
website will pick them up automatically. Until a file exists, the site
shows a labeled placeholder box so you always know what's missing.

WHAT TO ADD
------------------------------------------------------------
1. headshot.jpg
   -> /public/images/headshot.jpg
   -> Your portrait. Square, 600 x 600px or larger.
   -> Shows in the "About me" section.

2. projects/creative-dashboard.png
   -> /public/images/projects/creative-dashboard.png
   -> Screenshot of the Creative Operations Dashboard.
   -> 16:9 ratio, 1200 x 675px, PNG or WebP.

3. projects/teardown.png
   -> /public/images/projects/teardown.png
   -> Screenshot of TearDown.
   -> 16:9 ratio, 1200 x 675px, PNG or WebP.

4. og-image.png  (optional but recommended)
   -> /public/images/og-image.png
   -> The preview image people see when you paste your link in
      social media / chat. 1200 x 630px.

OPTIONAL
------------------------------------------------------------
- Roan-Uson-CV.pdf  ->  put in /public/  (one level up) to enable the
  "Download CV" button. To hide the button instead, set resumePdf to ""
  in content/portfolio.ts.

CHANGING FILE NAMES
------------------------------------------------------------
If you name a file differently, just update the matching path in
content/portfolio.ts (the `image`, `headshot`, or `ogImage` fields).
